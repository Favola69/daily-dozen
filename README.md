# daily-dozen
Plant food database and Daily Dozen recommendation generator created in Google Colab
Projekt: Daily Dozen növényi étrend generátor

Tartalom:
- napi_tucat_novenyi_adatbazis_napi_minimum_oszlopokkal.csv
- napitucatnovenyiadatbazis2.ipynb

Funkciók:
- adatelemzés
- hiányzó adatok ellenőrzése
- Daily Dozen kategóriák
- random napi ajánló
- makróösszesítés
- interaktív szűrések
